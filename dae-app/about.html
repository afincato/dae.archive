Welcome to the digital archive of Design Academy Eindhoven. We are currently in our first testing phase.

All users can have their own repositories inside the archive.  
All users with a repository can upload anything they want.
All users own the contents of their respective repositories.
Right now, the archive is stored on a server inside jbg's house, in Amsterdam.

The archive is based on git, a version control system that stores and logs different versions of a folder and its contents.

Every day at 02:05, an indexing script runs automatically, which creates and updates the metadata of all uploaded files. A file in our archive will typically have the following metadata: ID, name, path, author, extension, commit messages, and version.

If a file is an image or video, its metadata will also include the predominant colours, as well as a list of tags that describe the visual contents of the file, as read by TensorFlow's image tagging script, that is based on this dataset.

If a folder contains a file named "index.html", web browsers will render and display that webpage instead of the contents of the folder it's in.

The archive is extensible. Users are encouraged to design and develop tools and extensions of the archive by using its API. Some examples of this include jbg's colour picker and km's metadata navigator.
